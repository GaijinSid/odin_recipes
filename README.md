# odin_recipes
An HTML project to create a basic recipe website

This project is designed to put into practice recently learnt HTML skills. The key points that I have learned to understand and devlop are:

Creating the HTML Boilerplate
Formatting content using tags for 'Headers' and 'Paragrahs'
Creating both unordered and ordered lists
Understanding 'absolute' and 'relative' links and creating them where appropriate
Adding images and applying image attributes such as 'width' and 'alt'


Throughout this process I am also learning to connect my work with GitHub repositories and understand the importance of 'committing' work with accurate notes.
